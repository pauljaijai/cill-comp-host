# Minecraft 方塊包裝器

讓網頁元素變成方塊，在小小的網頁裡挖呀挖呀挖！ᕕ( ﾟ ∀。)ᕗ

## 使用範例

### 基本用法

按住左鍵挖掘元素，持續一段時間後會將元素挖走。

對洞口按右鍵則可將元素放回去。( ´ ▽ ` )ﾉ
